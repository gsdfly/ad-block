# 广告屏蔽脚本 - 支持via浏览器

## 功能特性

- **自动广告检测**：基于内置规则自动检测和屏蔽广告
- **动态广告检测**：使用MutationObserver监听DOM变化，检测动态加载的广告
- **手动选择功能**：允许用户手动选择广告元素并添加到屏蔽规则
- **Gitee集成**：将屏蔽规则保存在Gitee代码块中，实现跨设备同步

### 内置广告规则
脚本内置了常见的广告选择器：

- 常见广告ID：`#ad`, `#ads` 等
- 广告关键词：`[class*="ad"]`, `[id*="ad"]`
- 小说网站常见广告：`.chapter-ad`, `.content-ad` 等

## 使用方法

### 在via浏览器中使用
1. 打开via浏览器
2. 进入设置 → 脚本
3. 点击「添加脚本」
4. 输入脚本名称（如「广告屏蔽」）
5. 将 `ad-blocker.js` 的内容复制到脚本编辑器中
6. 保存脚本
7. 启用脚本

### 手动选择广告
1. 打开任意网页
2. 点击页面右上角的「选择广告」按钮
3. 鼠标移动到广告元素上，会显示红色边框
4. 点击广告元素，会自动添加到屏蔽规则并保存到Gitee
5. 点击「取消选择」按钮退出选择模式

### 查看屏蔽日志
在浏览器开发者工具的控制台中，可以看到屏蔽的广告元素和原因。

## 技术实现

- **广告检测**：使用CSS选择器和DOM操作
- **动态广告**：使用MutationObserver监听DOM变化
- **Gitee集成**：使用Gitee API进行规则的存储和读取
- **手动选择**：使用鼠标事件和元素选择器生成

## 注意事项

1. 首次使用时，需要确保Gitee代码块存在
2. 脚本需要网络权限以访问Gitee API
3. 某些网站可能会使用反广告屏蔽技术，可能需要手动添加规则
4. 频繁的DOM变化可能会影响性能，脚本已做了优化
